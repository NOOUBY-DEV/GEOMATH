# GEOMATH
A SIMPLE .JAVA CLASS WITH GEOMETRY METHODS FOR CALCULATION


- INCLUDES 2D & 3D SHAPES
- CALCULATES PROPERTIES OF SHAPES

* MADE IN ECLIPSE AS A JAVA PROJECT


TO USE :
1. DOWNLOAD GEOMATH.java
2. DIRECTLY COPY IT INTO YOUR PROJECT PACKAGE
3. IN GEOMATH.java, EDIT package >MAIN<; TO YOUR CORRESPONDING PACKAGE NAME (EG. package MYPACKAGE;)
4. FROM OTHER CLASSES (EG. MAIN SCRIPT), ACCESS IT VIA IMPORT STATMENT (EG. import MYPACKAGE.GEOMATH;)
5. THX ALOT FOR USING !
